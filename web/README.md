# Web dashboard (stretch goal)

A React + TypeScript live fleet map that subscribes to MQTT over WebSockets
(port 9001) and renders robot positions, task assignments, and paths in real time.

This is a stretch goal — Grafana already covers the KPIs. Build this only if ahead
of schedule; it is, however, a strong visual for the demo video and plays to your
React/TS strength. Keep it self-contained here.
